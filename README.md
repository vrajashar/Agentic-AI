# 🚀 Agentic AI Systems

This repository contains implementations of **Agentic AI workflows** using modern frameworks like **LangChain, LangGraph, and MCP (Model Context Protocol)**.

It focuses on building real-world AI systems involving **retrieval, reasoning, and multi-step decision making**.

---

## 📌 Projects

### 🧬 Biomedical RAG
A domain-specific **Retrieval-Augmented Generation (RAG)** system for biomedical data.
- Semantic search with embeddings
- Context-aware Q&A
- Reduced hallucination via retrieval pipelines

### 🌐 Multi-Source Agent
An **agent-based system** that gathers and reasons across multiple sources.
- Tool calling & API integration
- Multi-step reasoning (LangGraph)
- Dynamic workflow execution
